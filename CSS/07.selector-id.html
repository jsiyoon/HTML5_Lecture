id selector

id 속성(attribute)의 값으로 요소 선택

id속성(attribute) 작성 규칙

한 페이지 내에서 여러 요소가 같은 id를 가질 수 없음.

id 값 : 공백x, -,_,. 등은 호환 문제 일으킬 수 있음.

영문대소문자, 숫자, 조합, 숫자가 앞에 오면 안됨.
